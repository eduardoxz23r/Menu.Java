#  Sistema de Autenticação e Menu Interativo em Java

Este projeto foi desenvolvido como parte dos meus estudos iniciais em **Lógica de Programação** e **Linguagem Java**. O objetivo foi aplicar conceitos fundamentais de controle de fluxo, validação de dados e interação com o usuário via console.

### 🛠️ Funcionalidades
* **Autenticação de E-mail:** Validação de string utilizando o método `.equals()`.
* **Segurança de Senha:** Sistema que permite até 3 tentativas de acesso antes de bloquear o programa.
* **Menus Aninhados:** Navegação entre categorias (Comidas e Bebidas) com subopções específicas.
* **Validação de Opções:** O programa identifica entradas inválidas e solicita novos dados ao usuário.

### 💻 Tecnologias Utilizadas
* **Linguagem:** Java
* **IDE:** IntelliJ IDEA
* **Entrada de Dados:** `java.util.Scanner`

### 📖 O que eu aprendi neste projeto
Durante o desenvolvimento, pude praticar o uso de:
1. **Estruturas de Repetição:** Uso do `while` para criar loops de validação.
2. **Condicionais:** Aplicação de `if`, `else if` e operadores lógicos (`&&`, `||`).
3. **Gerenciamento de Fluxo:** Uso do comando `return` e `break` para encerrar o programa em casos de erro ou bloqueio.
